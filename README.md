--- Apache Web App com Docker Compose
Este projeto demonstra como criar uma aplicação web simples em HTML e executá-la dentro de um container Docker usando o servidor Apache (httpd).

--- Tecnologias Utilizadas
HTML: para criar a página web.
Docker: para criar e executar containers.
Docker Compose: para orquestrar o container Apache.
Apache HTTP Server: servidor web que hospeda a aplicação.

--- O que foi feito
Criação de uma página HTML com mensagem "Olá, mundo!".
Organização dos arquivos em uma estrutura clara.
Criação de um arquivo compose.yml com as definições do servidor Apache.
Execução da aplicação via Docker Compose.

--- Execute o container com Docker Compose:
bash
docker-compose up
